To run the program:
1. Open environment:
    source venv/bin/activate

2. (if you haven't) requirements
install pip install -r requirements.txt

3. Run server locally:
    python dev_server.py

---

NEW: Run with VS Code Debugger
1. Open this folder in VS Code.
2. Go to the "Run and Debug" tab (Ctrl+Shift+D).
3. Select "Python: Dev Server" and press the Green Play button (or F5).

NEW: Run with uv (Fast & Automatic)
1. Install uv (if needed): curl -LsSf https://astral.sh/uv/install.sh | sh
2. Run the server:
    uv run dev_server.py

    


---

TROUBLESHOOTING:
If VS Code shows "Import 'livereload' could not be resolved":
1. Press Ctrl+Shift+P (or F1) in VS Code.
2. Type "Python: Select Interpreter".
3. Select the environment named 'venv' (ensure you have run `pip install livereload` inside it).
